# random-forest

## **Installation**
### **1. Ubuntu/MacOS**
```
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
### **2. Windows**
```
python -m venv .venv
& .venv/Scripts/Activate.ps1
pip install -r requirements.txt
```

## **How to run**
Run this command and see result in `tree.json`
```
python main.py
```
